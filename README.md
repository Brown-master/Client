# 견인차량들 사이의 간의 경쟁에서 나오는 불법행위를 줄이기 위한 어플리케이션

## Decritpion
견人은 견인 차량 운전자들을 위한 어플리케이션으로 도로에서 발생하는 사고를 실시간으로 분석하고 사고메시지로 알려준다. 
사고 현장에  도착하기 전 견인차량들 간의 우선순위를 두어서 견인 차량들간의 2차사고나 불법행위를 예방한다.
